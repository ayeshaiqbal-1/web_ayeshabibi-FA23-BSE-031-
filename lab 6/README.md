## Getting Started

To view any of these HTML files:

1. Open the desired HTML file in a web browser
2. Alternatively, use a local web server for proper rendering:
   ```bash
   # Using Python (if installed)
   python -m http.server 8000
   
   # Using Node.js (if installed) 
   npx http-server
   ```
