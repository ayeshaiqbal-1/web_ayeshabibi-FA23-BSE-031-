# Lab 7

This folder contains small HTML/JavaScript exercises demonstrating browser timers.

## Contents
- A1.html — setTimeout to show a message after 3 seconds
- A2.html — start/stop controls using setTimeout and clearTimeout
- A3.html — digital clock using setInterval, with a stop button
- task.html — supplemental exercise/starter

## How to Run
- Open any file directly in a modern browser (Chrome, Edge, Firefox).
- Or use a local server/Live Server extension to serve this folder.

## Notes
- No build step required; files are pure HTML/JS.
- Tested with current desktop browsers.
